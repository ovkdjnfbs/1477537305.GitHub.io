DeepSeek 学习备注

一、基础概念层

  1. 向量运算
  • 点积（内积）
  • 公式：a·b = Σ(aᵢbᵢ)
  • 应用：计算夹角、投影（图形学光照计算）
  • 叉积（三维）
  • 公式：a×b = [a₂b₃-a₃b₂, a₃b₁-a₁b₃, a₁b₂-a₂b₁]
  • 应用：求平面法向量（3D建模）
  1. 矩阵基本操作
  • 乘法规则
    Cᵢⱼ = Σ(AᵢₖBₖⱼ)
  • 关键：仅当A列数=B行数时可乘
  • 转置
    (Aᵀ)ᵢⱼ = Aⱼᵢ
  • 应用：神经网络权重更新

 

二、核心理论层

  1. 矩阵求逆
  • 可逆条件：det(A) ≠ 0
  • 2×2矩阵公式：
    text
    A⁻¹ = (1/det(A)) * [d -b; -c a]
  • 应用：解线性方程组 Ax = b → x = A⁻¹b
  1. 行列式
  • 几何意义：线性变换的缩放因子
  • 计算方法：
  • 2×2：det(A) = ad - bc
  • 3×3：对角线法则（Sarrus规则）
  1. 特征值与特征向量
  • 定义：Av = λv
  • 求解步骤：
  1. 解特征方程 det(A - λI) = 0
  2. 对每个λ求(A - λI)v = 0
  • 应用：PCA降维、物理系统稳定性分析

 

三、几何应用层

  1. 线性变换
  • 旋转矩阵（2D）：
    text
    [cosθ -sinθ]
    [sinθ  cosθ]
  • 缩放矩阵：
    text
    [sx  0]
    [0  sy]
  1. 齐次坐标（图形学关键）
  • 3D点表示为[x, y, z, 1]
  • 变换矩阵示例（平移）：
    text
    [1 0 0 tx]
    [0 1 0 ty]
    [0 0 1 tz]
    [0 0 0 1]

 

四、数值计算层

  1. 矩阵分解
  • LU分解：解方程组的高效方法
  • QR分解：最小二乘拟合
  • SVD分解：推荐系统、图像压缩
  1. 稀疏矩阵优化
  • CSR存储格式：
  • 非零值数组：data = [a, b, c, d]
  • 行指针：row_ptr = [0, 2, 3]
  • 列索引：col_idx = [0, 1, 0]

 

五、现代应用领域

  1. 计算机图形学
  • MVP矩阵：Model × View × Projection
  • 法线变换：(M⁻¹)ᵀ·n
  1. 机器学习
  • 梯度下降：w = w - α(XᵀX)⁻¹Xᵀy
  • 卷积运算：本质是稀疏矩阵乘法
  1. 量子计算
  • 量子门操作表示为酉矩阵（U†U = I）

 

六、关键公式速查表

概念 公式 应用场景
点积 a·b = |a||b|cosθ 光照计算
叉积模长 |a×b| = |a||b|sinθ 计算平行四边形面积
矩阵迹 tr(A) = ΣAᵢᵢ 特征值求和
伴随矩阵 adj(A) = Cᵀ (C为余子式矩阵) 求逆矩阵

 

七、学习路径建议

  1. 入门：掌握2×2/3×3矩阵运算
  2. 进阶：理解秩、线性空间、基变换
  3. 高阶：学习张量运算（深度学习框架底层）

这些知识点构成了从数学基础到工程应用的完整链条，建议通过实际编程（如NumPy/SciPy）深化理解。
